# Cozy Cast Avatars

Composited badges der 8 CozyQuiz-Teams im CozyWolf-Brand-Style.

## Naming convention
Pattern: `avatar-{slug}-badge-1024.png`

Der Design-Lab Ring-Compositor (→ `/design-lab.html` → Section 1.7) produziert
diese Dateien exakt benannt, wenn du den Download-Button klickst.

| Slug        | Team             | Hex Body   | Hex Hoodie |
| ----------- | ---------------- | ---------- | ---------- |
| `waschbaer` | Waschbär — Teal  | `#14B8A6`  | `#F59E0B`  |
| `faultier`  | Faultier — Lime  | `#84CC16`  | `#7C2D12`  |
| `pinguin`   | Pinguin — Blue   | `#2563EB`  | `#FDE047`  |
| `kuh`       | Kuh — Orange     | `#F97316`  | `#581C87`  |

| `shiba`     | Shiba — Pink     | `#EC4899`  | `#0EA5E9`  |

## Usage
Die Palette-Preview in `/design-lab.html` (Section 1.5) lädt diese Dateien
automatisch und zeigt sie auf Dark- und Light-Hintergrund. Fehlt eine Datei,
fällt die Preview still auf Emoji-Platzhalter zurück.
