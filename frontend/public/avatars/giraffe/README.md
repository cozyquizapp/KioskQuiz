# Giraffe Avatar - 4 States

Bitte hier die 4 SVG-Dateien für die Giraffe platzieren:

- **gehen.svg** - Giraffe läuft (Standard/Walking)
- **schauen.svg** - Giraffe schaut/denkt nach (Looking/Thinking)
- **freuen.svg** - Giraffe freut sich (Happy/Celebrating)
- **weinen.svg** - Giraffe ist traurig (Sad/Crying)

## Verwendung
Diese SVGs werden automatisch basierend auf dem State geladen.
