# Katze Avatar - 4 States

Bitte hier die 4 SVG-Dateien für die Katze platzieren:

- **gehen.svg** - Katze läuft (Standard/Walking)
- **schauen.svg** - Katze schaut/denkt nach (Looking/Thinking)
- **freuen.svg** - Katze freut sich (Happy/Celebrating)
- **weinen.svg** - Katze ist traurig (Sad/Crying)

## Verwendung
Diese SVGs werden automatisch basierend auf dem State geladen.
