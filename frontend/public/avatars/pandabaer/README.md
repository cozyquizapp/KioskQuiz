# Pandabär Avatar - 4 States

Bitte hier die 4 SVG-Dateien für den Pandabär platzieren:

- **gehen.svg** - Pandabär läuft (Standard/Walking)
- **schauen.svg** - Pandabär schaut/denkt nach (Looking/Thinking)
- **freuen.svg** - Pandabär freut sich (Happy/Celebrating)
- **weinen.svg** - Pandabär ist traurig (Sad/Crying)

## Verwendung
Diese SVGs werden automatisch basierend auf dem State geladen.
