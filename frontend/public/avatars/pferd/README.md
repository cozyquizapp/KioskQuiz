# Pferd Avatar - 4 States

Bitte hier die 4 SVG-Dateien für das Pferd platzieren:

- **gehen.svg** - Pferd läuft (Standard/Walking)
- **schauen.svg** - Pferd schaut/denkt nach (Looking/Thinking)
- **freuen.svg** - Pferd freut sich (Happy/Celebrating)
- **weinen.svg** - Pferd ist traurig (Sad/Crying)

## Verwendung
Diese SVGs werden automatisch basierend auf dem igelState geladen.
Avatar ID muss auf `avatar11` oder entsprechendes Pferd-Avatar gesetzt sein.
