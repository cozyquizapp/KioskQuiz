# Wolf Avatar - 4 States

Bitte hier die 4 SVG-Dateien für den Wolf platzieren:

- **gehen.svg** - Wolf läuft (Standard/Walking)
- **schauen.svg** - Wolf schaut/denkt nach (Looking/Thinking)
- **freuen.svg** - Wolf freut sich (Happy/Celebrating)
- **weinen.svg** - Wolf ist traurig (Sad/Crying)

## Verwendung
Diese SVGs werden automatisch basierend auf dem State geladen.
