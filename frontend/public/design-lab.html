🎛️ Stream Deck Preset

Angepasst an die App-Palette (Blue/Green/Red/Orange + Amber-Accent). Klicke „PNG 144px" oder „PNG 288px" um die Datei sofort zu downloaden und ins Stream-Deck-Profil zu ziehen.

So baust du's ein: Stream Deck App → Profil → Key auswählen → rechts „Titel löschen" + „Bild" auf die PNG-Datei setzen.
Stream Deck 6.5+ unterstützt auch SVG direkt (unter /streamdeck/ abgelegt). Für ältere Versionen PNG nehmen — 288px für XL-Decks, 144px reicht für Classic & Plus.

✨ Lottie Celebration Test

Probier aus, wie animierte Lottie-Effekte über einem Team-Avatar wirken (z. B. beim Winner-Reveal). Hol dir eine .json-Datei bei LottieFiles, ziehe sie ins Feld oder pick sie aus — sie spielt live über dem Beispiel-Pill. Ohne Upload läuft unten ein simpler CSS- Sparkle als Platzhalter.
